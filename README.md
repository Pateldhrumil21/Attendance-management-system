# 📋 Attendance Management System

## 📌 Project Overview

The Attendance Management System is a Python-based project developed to manage and analyze student attendance records efficiently. The system helps track attendance, identify low-attendance students, and generate attendance-related insights.

## 🚀 Features

- Calculate total attendance records
- Count present days for each student
- Count absent days for each student
- Add new student records
- Delete existing student records
- Identify students with low attendance
- Generate monthly attendance reports
- Find students with highest attendance
- Calculate attendance percentage
- Sort students based on attendance

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy

## 📊 Analysis Performed

### Student-Level Analysis

- Total Present Days
- Total Absent Days
- Attendance Percentage
- Highest Attendance Student
- Low Attendance Students

### Attendance Management

- Add New Student
- Delete Student
- Update Attendance Records
- Monthly Attendance Tracking
